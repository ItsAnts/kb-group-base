Задача: Учёт заказов клиентов

Описание ситуации

Есть интернет-магазин.
Каждый заказ хранится как кортеж, содержащий фиксированный набор данных.

Структура заказа:

(id_заказа, имя_клиента, сумма_заказа, город)

Пример одного заказа:

order = (1001, "Иван", 2500, "Москва")

Все заказы хранятся внутри общего кортежа orders.

Пример:

orders = (
    (1001, "Иван", 2500, "Москва"),
    (1002, "Анна", 4200, "Казань"),
    (1003, "Петр", 1800, "Москва"),
    (1004, "Ольга", 5600, "Самара"),
    (1005, "Мария", 3100, "Москва")
)

⸻

Цель работы

Отработать:
 • индексацию кортежей
 • вложенные кортежи
 • перебор кортежей
 • распаковку кортежей
 • методы count() и index()
 • работу с неизменяемыми структурами

⸻
