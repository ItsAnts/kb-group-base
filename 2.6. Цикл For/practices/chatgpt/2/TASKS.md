Уровень 1 — Легкий (3 задачи)

Задача 1 — Подсчет общей выручки за день

Контекст:
Есть список продаж интернет-магазина за день. Каждое число — сумма одной покупки в евро.

Входные данные:

sales = [120, 85, 230, 45, 99]

Задача:
С помощью цикла for:
	1.	Посчитать общую выручку за день
	2.	Вывести итоговую сумму

Ожидаемый результат:

Total revenue: 579

Уровень 4 — Тяжело (1 задача)

Задача 10 — Аналитика продаж по неделям

Контекст:
Есть продажи по дням за несколько недель.
Каждая неделя — отдельный список.

Входные данные:

weeks = [
    [120, 340, 560, 230, 150],
    [90, 80, 300, 400, 500],
    [600, 700, 200, 100, 50]
]

Задача:

С помощью вложенных циклов for:
	1.	Посчитать сумму продаж по каждой неделе
	2.	Найти неделю с максимальной выручкой
	3.	Вывести:
	•	суммы по неделям
	•	номер недели с максимальной выручкой
	•	максимальную сумму

Ожидаемый результат:

Week sums: [1400, 1370, 1650]
Best week: 3
Max revenue: 1650
