Уровень 1 — Легкий (3 задачи)

Задача 1 — Подсчёт выручки за день

Компания продаёт товары. Нужно посчитать общую выручку за день.

Вводные данные:
	•	В первой строке число n — количество продаж за день
	•	Далее n строк — сумма каждой продажи (в евро)

Задача:

Используя цикл for:
	•	просуммировать все продажи
	•	вывести общую выручку

Пример ввода:

5
120
85
40
230
75

Ожидаемый вывод:

550

Уровень 4 — Тяжело (1 задача)

Задача 10 — Расчёт месячной прибыли с налогами

Компания анализирует прибыль по месяцам.

Правила:
	•	если прибыль > 10000 → налог 20%
	•	если прибыль от 5000 до 10000 → налог 10%
	•	если < 5000 → налог 0%

Вводные данные:
	•	n — количество месяцев
	•	Далее n строк — прибыль по каждому месяцу

Задача:

Используя for:

Для каждого месяца:
	1.	рассчитать налог
	2.	рассчитать чистую прибыль
	3.	вывести чистую прибыль
	4.	после цикла вывести:

	•	общую чистую прибыль за весь период

Пример ввода:

3
12000
8000
4000

Ожидаемый вывод:

9600
7200
4000
20800
