Уровень 1 — Базовый (while + списки)

Задача 1 — Обработка списка заказов

Описание:
Есть список заказов. Нужно обработать каждый заказ по очереди.

orders = [1200, 3400, 560, 2300, 780]

Требуется:
	1.	Использовать while
	2.	Пройти по списку по индексам
	3.	Вывести сумму каждого заказа
	4.	Посчитать общую сумму заказов
	5.	Вывести итоговую сумму
