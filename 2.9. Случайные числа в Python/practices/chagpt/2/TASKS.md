Задача 1: Доход кофейни за рабочую неделю

Контекст

Кофейня продаёт чашки кофе.
Количество проданных чашек каждый день случайное.

Вводные данные

- Дней: 6
- Продажи в день: 30–80 чашек
- Цена одной чашки: 4.5 €

Задача

1. Сгенерировать количество проданных чашек на каждый день.
2. Сохранить данные в список.
3. Рассчитать:
   - выручку за каждый день
   - общую выручку
   - день с максимальными продажами

Использовать

- random.randint
- список
- цикл for

Задача 6: Управление складскими заказами

Контекст

Склад принимает и отправляет заказы.
Каждый заказ содержит случайное количество товаров.

Вводные данные

Товары:

items = {
"Phone": 600,
"Tablet": 400,
"Smartwatch": 250,
"Camera": 750,
"Speaker": 150
}

Описание:

- ключ — товар
- значение — цена

Параметры:

- Заказов: 20
- Количество единиц товара в заказе: 1–5

Структура заказа

[
{
"item": "Phone",
"quantity": 3,
"total_price": 1800
}
]

Задача

1. Для каждого заказа:
   - выбрать случайный товар
   - сгенерировать количество
   - рассчитать стоимость заказа
2. Сохранить все заказы в список.
3. Рассчитать:
   - общую выручку
   - самый дорогой заказ
   - сколько заказов по каждому товару
   - средний чек

Использовать

- random.choice
- словари
- списки
- вложенные структуры
