Большая задача на input / print / базовые вычисления

Задача: Анкета пользователя

Написать программу, которая собирает данные о пользователе и выводит оформленную анкету.

⸻

Требования

Программа должна запросить у пользователя:
	1.	Имя
	2.	Фамилию
	3.	Возраст
	4.	Город проживания
	5.	Любимый предмет
	6.	Оценку по любимому предмету
	7.	Рост (в сантиметрах)
	8.	Вес (в килограммах)

Все данные вводятся через input().

Возраст, рост и вес должны быть преобразованы в int.

⸻

Дополнительные вычисления

Программа должна:
	1.	Посчитать возраст через 5 лет
	2.	Посчитать рост в метрах
(разделить сантиметры на 100)
	3.	Вывести примерный индекс массы тела (ИМТ):

ИМТ = вес / (рост_в_метрах ** 2)


⸻

Формат вывода

Вывести анкету в следующем формате:

==============================
АНКЕТА ПОЛЬЗОВАТЕЛЯ
==============================

Имя: Иван
Фамилия: Иванов
Город: Москва

Возраст: 16 лет
Через 5 лет: 21 лет

Любимый предмет: Математика
Оценка: 5

Рост: 170 см (1.7 м)
Вес: 65 кг

Индекс массы тела: 22.49

==============================


⸻

Что проверяет задача
	•	работа с input()
	•	работа с print()
	•	преобразование типов (int, float)
	•	форматирование вывода
	•	базовые вычисления
	•	переменные
	•	структура программы

⸻

Если нужно — могу написать:
	•	вариант ещё сложнее
	•	вариант для экзамена
	•	вариант с условиями (if)
	•	вариант в стиле реального проектаi
