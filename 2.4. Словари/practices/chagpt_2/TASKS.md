Уровень 1 — Базовый (основа словарей)

Задача 1 — Балансы счетов компании

Описание:
В компании ведется учет денежных остатков по банковским счетам.

accounts = {
    "Основной": 250000,
    "Резервный": 120000,
    "Зарплатный": 98000
}

Требуется:
	1.	Вывести все счета и их остатки
	2.	Найти остаток по счету "Резервный"
	3.	Добавить новый счет "Инвестиционный" с балансом 50000
	4.	Удалить счет "Зарплатный"
	5.	Проверить, есть ли счет "Кредитный"

Задача 8 — Сегментация товаров по продажам

Описание:
Компания делит товары на категории по популярности.

    "Ноутбук": 35,
    "Мышь": 120,
    "Клавиатура": 65,
    "Монитор": 20
}

Требуется:

Создать новый словарь:

segments = {
    "Хит": [],
    "Средние": [],
    "Низкие": []
}

Правила:
	•	"Хит" — больше 100 продаж
	•	"Средние" — 30–100
	•	"Низкие" — меньше 30

Заполнить словарь segments.
