Тема: словари (dict) в Python
Контекст: бизнес-задачи (клиенты, продажи, склад, финансы)
Формат: 3 уровня — базовый, средний, продвинутый

⸻

Уровень 1 — Базовый (работа с ключами и значениями)

Задача 1 — База клиентов

Описание:
Есть словарь с клиентами и их балансом. Нужно вывести информацию по каждому клиенту.

clients = {
    "Иван": 12000,
    "Мария": 8500,
    "Олег": 23000
}

Требуется:
	1.	Вывести всех клиентов и их баланс
	2.	Найти баланс клиента "Мария"
	3.	Добавить нового клиента "Анна" с балансом 15000
	4.	Удалить клиента "Олег"

⸻

Задача 2 — Каталог товаров

Описание:
Есть словарь с товарами и их ценами.

products = {
    "Ноутбук": 75000,
    "Мышь": 1200,
    "Клавиатура": 3500
}

Требуется:
	1.	Вывести список всех товаров
	2.	Найти самый дорогой товар
	3.	Увеличить цену "Мышь" на 10%
	4.	Проверить, есть ли товар "Монитор"

⸻

Задача 3 — Продажи за день

Описание:
Есть словарь с количеством проданных товаров.

sales = {
    "Ноутбук": 5,
    "Мышь": 18,
    "Клавиатура": 9
}

Требуется:
	1.	Посчитать общее количество проданных товаров
	2.	Найти товар с максимальными продажами
	3.	Добавить продажи "Монитор": 7

⸻

Уровень 2 — Средний (вложенные словари)

Задача 4 — Профили сотрудников

Описание:
Каждый сотрудник имеет должность и зарплату.

employees = {
    "Иван": {
        "position": "Менеджер",
        "salary": 70000
    },
    "Мария": {
        "position": "Аналитик",
        "salary": 85000
    }
}

Требуется:
	1.	Вывести должность "Мария"
	2.	Увеличить зарплату "Иван" на 15%
	3.	Добавить нового сотрудника
	4.	Вывести всех сотрудников с зарплатой выше 80000

⸻

Задача 5 — Склад товаров

Описание:
Есть склад с количеством и ценой товаров.

warehouse = {
    "Ноутбук": {
        "price": 75000,
        "quantity": 4
    },
    "Мышь": {
        "price": 1200,
        "quantity": 25
    }
}

Требуется:
	1.	Посчитать общую стоимость склада
	2.	Найти товар с минимальным количеством
	3.	Добавить новый товар
	4.	Уменьшить количество "Мышь" на 3

⸻

Задача 6 — История заказов

Описание:
Каждый клиент имеет список заказов.

orders = {
    "Иван": [1200, 3500, 800],
    "Мария": [5000, 2300],
    "Олег": [700]
}

Требуется:
	1.	Посчитать сумму заказов для каждого клиента
	2.	Найти клиента с максимальной суммой заказов
	3.	Добавить новый заказ "Иван": 4500

Уровень 3 — Продвинутый (логика и аналитика)

Задача 7 — Финансовый отчет

Описание:
Компания ведет учет доходов по месяцам.

finance = {
    "Январь": 120000,
    "Февраль": 95000,
    "Март": 143000,
    "Апрель": 87000
}

Требуется:
	1.	Найти месяц с максимальным доходом
	2.	Найти месяц с минимальным доходом
	3.	Посчитать средний доход
	4.	Определить месяцы с доходом ниже среднего

⸻

Задача 8 — Аналитика клиентов

Описание:
Есть база клиентов с количеством покупок.

customers = {
    "Иван": 12,
    "Мария": 5,
    "Олег": 19,
    "Анна": 7
}

Требуется:
	1.	Разделить клиентов на категории:

	•	"VIP" — больше 15 покупок
	•	"Обычные" — 5–15
	•	"Новые" — меньше 5

	2.	Создать новый словарь:

segments = {
    "VIP": [...],
    "Обычные": [...],
    "Новые": [...]
}

Задача 9 — Полная система учета продаж

Описание:
Компания продает товары. Есть данные по продажам.

sales_data = {
    "Ноутбук": {
        "price": 75000,
        "sold": 5
    },
    "Мышь": {
        "price": 1200,
        "sold": 18
    },
    "Клавиатура": {
        "price": 3500,
        "sold": 9
    }
}

Требуется:
	1.	Посчитать выручку по каждому товару
	2.	Посчитать общую выручку
	3.	Найти самый прибыльный товар
	4.	Отсортировать товары по выручке
	5.	Создать отчет вида:

{
    "Ноутбук": 375000,
    "Мышь": 21600,
    "Клавиатура": 31500
}
